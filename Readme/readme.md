# E-Commerce Sales & Customer Analytics

## Project Overview
End-to-end data analysis of the Olist Brazilian E-Commerce dataset using Python, MySQL, and Tableau.

## Tools Used
- Python (pandas, matplotlib, seaborn, sqlalchemy)
- MySQL (data storage + SQL analysis)
- Tableau (interactive dashboards)

## Dataset
- Source: Olist E-Commerce Public Dataset (Kaggle)
- Tables: customers, orders, payments, order_items, reviews, products
